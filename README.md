# AIM

To write a Python program using OpenCV to capture an image from the webcam and perform video processing operations such as saving, displaying, resizing, and rotating the video frames.

# SOFTWARE REQUIRED

Python 3.7 (Anaconda)
Jupyter Notebook (for interactive development and execution)

# EXPLANATION

Webcam image and video processing involves capturing live frames from a camera device and applying different operations to the frames in real time. OpenCV provides functions to capture video, save images, resize frames, rotate video streams, and display processed outputs efficiently.

# ALGORITHM

Step 1: Import the required libraries and initialize the webcam using cv2.VideoCapture()

Step 2: Capture frames continuously from the webcam

Step 3: Save a frame as a JPG image using cv2.imwrite()

Step 4: Display the live video stream using cv2.imshow()

Step 5: Resize the video frame and display the resized output

Step 6: Rotate the video frame using OpenCV functions and display the rotated output

# RESULT

Thus, the webcam image was successfully captured and various video processing operations such as saving, displaying, resizing, and rotating were performed using OpenCV in Python.
